# Weather Background Assets

Please add your GIF or Image files here with the following names. The application will automatically load them based on the weather condition.

- **clear.gif** - For sunny/clear skies
- **clouds.gif** - For cloudy weather
- **rain.gif** - For rainy weather
- **snow.gif** - For snowy weather
- **mist.gif** - For fog/mist/haze
- **thunderstorm.gif** - For stormy weather
- **default.gif** - Fallback background

> **Note:** You can use .jpg or .png if you prefer, but you'll need to update the file extension in `js/app.js` if you do so. Currently configured for `.gif`.
